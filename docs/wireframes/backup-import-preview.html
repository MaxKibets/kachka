Edge cases · related states

  • Older schema → auto-migrate before preview (silent, per spec §13.6). Preview показує мігровану версію
  • Newer schema → не доходить до цього screen-у; error sheet на попередньому кроці
  • Validation fail (corrupted JSON) → error sheet, не доходить сюди
  • Atomic transaction fail при applying → rollback + error sheet (per spec §13.3 step 7)
  • Merge mode з 0 нових entities (всі duplicates) → toast Nothing to import на success (TBD як explicit state)
