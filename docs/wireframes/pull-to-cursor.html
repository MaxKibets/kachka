Opens from

  • In-workout — auto-scroll призупиняється при manual scroll, активний сет повністю поза viewport (per spec §5.8)

Opens to

  • Tap chip → smooth scroll до active set, chip ховається, auto-scroll re-engages
  • Юзер скролить так, що active set знов у viewport → chip ховається без tap
  • Все workout completed → chip ховається назавжди

Interactions

  • Visibility: chip з'являється коли cursor повністю поза viewport (з threshold 1-2 рядки)
  • Arrow direction: ↑ (up) коли cursor вище viewport (юзер проскролив униз), ↓ (down) коли нижче (юзер проскролив угору)
  • Лейбл: Set N · Exercise для standalone, A · Set N · Exercise для group (letter prefix per spec §5.8)
  • Тап → smooth scroll з easing, chip animates out
  • Re-engage cycle: після tap auto-scroll знову активний; manual scroll знову призупиняє; cursor exits → chip знову

Edge cases · related states

  • Cursor у viewport → chip прихований
  • Workout completed → chip прихований назавжди
  • Group cursor — letter chip префікс (TBD у production: tinted letter перед Set 3)
  • Rest timer running одночасно — БЕЗ колізії: rest = bottom bar's rest mode (§5.1/§5.10), chip плаває над баром. Різні шари, обидва видимі одночасно (chip зверху, rest-бар знизу). Design-review F3.1 resolved
  • Failed reps — cursor пропускає сет (per spec §5.7), chip посилається на наступний pending
  • Skipped exercise — cursor пропускає вправу (per spec §5.6), chip посилається на наступну з pending сетами

Notes

  • Chip — pill-shape, surface-1 фон, 0.5px mallard-tint border (rgba mallard 0.5), gentle shadow для elevation
  • Set number у Plex Mono Medium, mallard color (per visual §3.2 — числа mono, primary color сигналізує "позитивна важливість" як на in-progress banner)
  • Exercise name у text.primary, divider між number і name через ·
  • Іконка ↑ або ↓ — ліворуч, mallard color
  • Position: 92px from bottom (above action bar). Center horizontal — не торкається edges
