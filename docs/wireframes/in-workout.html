Opens from

  • Workout Builder → tap Start workout (builder.html — TBD у Batch 2)
  • Today banner → tap Resume (today-in-progress.html)

Opens to

  • Tap on set kg / reps tile → custom numpad sheet (numpad.html)
  • Tap on set number → set actions sheet (set-actions-sheet.html)
  • Tap on ✓ → save set + advance cursor + start rest timer (rest-timer.html)
  • Tap exercise ⋮ → exercise actions sheet (TBD — Insert after / Skip / Remove / Add to superset / Add note)
  • Tap top-bar ⋮ → workout actions sheet (TBD — Add exercise / Reorder / Discard)
  • Tap ← back → закриває modal без discard, банер на Today (today-in-progress.html)
  • Tap Finish workout → повноекранний completion-screen.html (≥1 set logged; 0 sets → одразу Discard confirm per spec §9.1)
  • Cursor exits viewport → return chip (pull-to-cursor.html)

Interactions

  • Vertical scroll. Auto-scroll keeps active set in view; manual scroll override pauses auto-scroll (per spec §5.8)
  • Tap ✓ on active set = save + advance cursor + start rest. Closing last set of last exercise = banner suggesting Finish workout
  • Long-press на set row = secondary trigger для set actions (per spec §8.1)
  • Active set: kg/Reps tiles тапабельні незалежно (numpad перемикає focus між полями)
  • Без bottom tab bar — modal full takeover (per spec §1, §2.2)

Edge cases · related states

  • З суперсетами — in-workout-with-supersets.html
  • Numpad open поверх → numpad.html
  • Set actions sheet → set-actions-sheet.html
  • Rest timer running → rest-timer.html
  • Cursor поза viewport → pull-to-cursor.html
  • Skipped exercise — TBD визуально (per spec §5.6: лишається в структурі помічена Skipped)
  • Failed reps (0) — TBD візуально (per spec §5.7 + visual §5.1)
  • Завершена вправа — collapsible до one-line summary (per spec §5.4) — TBD detail у наступних мокапах

Notes

  • Header — назва + edit-pencil + meta-рядок під нею + amber timer chip + ⋮. Timer chip — workout elapsed (per visual §5.6)
  • Set row grid — 32px | 1fr | 78px | 60px | 40px з 8px gaps, 14px padding (per visual §4.1)
  • Done state — opacity 0.55, mallard ✓ circle. Active state — top/bottom amber borders, surface-2 фон, surface-3 tile inputs, amber empty circle. Pending — text.tertiary values, 1px subtle empty circle (per visual §5.1)
  • Active row "ковтає" border-divider зверху і знизу через negative margin — щоб не було подвійних ліній (subtle divider + amber border)
