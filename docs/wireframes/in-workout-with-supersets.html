  • Tap на set (kg/reps tile) → numpad (numpad.html)
  • Tap на set number → set actions sheet (set-actions-sheet.html)
  • Tap ✓ → save + advance cursor (миттєво на A2 set 2 без rest, бо це ще mid-round per spec §6.5) → потім rest після set 2 of A2 (rest-timer.html)

Interactions

  • Cursor cycling: A1 set 2 done → миттєвий перехід на A2 set 2, без rest. Після A2 set 2 done → стартує rest 90s, наступний цикл — A1 set 3 (per spec §6.5)
  • Round dots оновлюються тільки коли всі вправи раунду закриті
  • Group active row — bleed по блоку вправ: впритул від mallard-рейки ліворуч до правого краю картки, top/bottom amber borders, без round corners (per visual §5.2)
  • Рейка не «обтікається» активним рядком — рейка лишається лівою рамкою блоку вправ, активний рядок стартує впритул до неї (margin: 0, без компенсацій)

Edge cases · related states

  • Без групи → in-workout.html
  • Rest між сетами групи → rest-timer.html
  • Numpad active → numpad.html
  • Set actions → set-actions-sheet.html
  • Pull-to-cursor chip → pull-to-cursor.html (для group cursor префікс A · Set 2 · Dumbbell row)
  • Ungroup mid-workout → set numbers стають sequential (per spec §6.7) — рендер як standalone exercise card
  • Group з 3+ вправами (B, C) — інлайн letter chip кольору ротації (B = ochre, C = plum, per visual §2.5), читається Superset B / Superset C. Не показано в цьому мокапі

Notes

  • Letter chip A = dusty blue #5C7B95 per visual §2.5 — маленький інлайн-чіп на рядку тайтлу (Superset A · Round 2 of 3), не окрема колонка. Child exercises без per-row літери — порядок задає позиція. Крос-реф A1/A2 лишається у History + компактних лейблах (rest bar A · Rest, return-to-cursor A · Set 2 · Dumbbell row)
  • Group card = surface-1 + 0.5px subtle border з усіх боків. 3px mallard-рейка — лише вздовж вправ, стартує під хедером (обгортка .group-children), трохи зсуває рядки й відмежовує їх від хедера (per visual §5.3)
  • Round indicators (●●○): першa точка зелена-A (round 1 done), друга amber (current round 2), третя приглушена-A (pending). Альтернатива — всі три у letter color з opacity варіаціями. Поки amber для current round щоб корелювало з amber active set
  • Bench press показано як collapsed completed card (per spec §5.4) — single row, mallard ✓ + summary
  • Rest 90s — інлайн у group meta. На bottom bar під час running rest — окремий dock (TBD у rest-timer.html)
  • Active wrap спрощено: рейка тепер на .group-children, тож активний рядок просто margin: 0 (flush до рейки) — компенсаційний padding-left у номера сета більше не потрібен

Soft start · pending (pre-start)

  • Цьому live-екрану передує pending (pre-start) стан In-workout (in-workout-pending.html, per spec §5.9) — усі сети pending, cursor вимкнено, clock не йде
  • Soft start: тренування «починається» на першому залогованому сеті АБО при явному Begin — що раніше. Тоді вмикається cursor, армується rest timer, стартує duration clock
  • Reps дочірніх вправ групи конфігуряться у pending-стані (один template × rounds, з prev орієнтиром) — див. group A у in-workout-pending.html. Rounds + rest — у superset-config-sheet.html
