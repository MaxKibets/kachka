  • Version 1.0.0 у Plex Mono (per visual §3.2 — числа в mono)
  • Report an issue + Open source licenses рядки прибрано (design-review F5.4): About = version + Source code + Privacy, точно як spec §12.5. Acknowledgements / licenses потребують фіксованого стеку + списку залежностей → v1.x
  • Privacy note inline як rendered text (не sub-screen) — короткий, безпосередньо читається (per spec §12.5 wording)
  • Footer-meta з author credit — м'який touch (не критичний для v1)
