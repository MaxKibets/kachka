Opens from

  • Today (today-has-history.html) → Repeat last (з prefilled списком, тут показано саме такий стан)
  • Today → Build from scratch (порожній список + chips)
  • Today (today-first.html) → Build workout (порожній список + chips, first launch)
  • History picker → tap на тренування → клонується (history-picker.html — TBD у Batch 4)

Opens to

  • Tap chip → додає вправу без сетів (кількість сетів/повторень налаштовується пізніше у In-workout pending, per spec §4.3 + §5)
  • Tap + Add exercise → exercise picker у Add mode (exercise-picker-add.html)
  • Tap row ⋮ → builder-row-menu-sheet.html
  • Tap Add to superset у row menu → партнер picker → superset-config-sheet.html
  • Tap Start workout → In-workout у pending state (pre-start). Годинник тривалості не запускається на тап — soft start: відлік стартує з першого залогованого сету або явного Begin (per spec §5.9). Live-стан: in-workout.html
  • Tap × close (або swipe-down) → confirmation sheet (Discard workout setup?) → Today (per spec §4.8)

Interactions

  • Drag handle (≡ зліва) — long-press grab → reorder вправ. Картка піднімається trace-ефектом (TBD у impl)
  • Tap pencil біля назви — inline edit поля з системною клавіатурою
  • Quick-add chips — горизонтальний scroll (overflow-x), 7 штук, fades справа якщо overflow
  • Свайп вниз від header — закриває builder з confirmation якщо щось змінено (per spec §4)
  • Start workout disabled поки список порожній (тут active бо 3 вправи)

Notes

  • Modal full takeover — без bottom tab bar (per spec §2.3). Bottom — лише sticky Start workout. Вихід — × close (modal convention, як exercise-picker-add §11) або swipe-down, обидва через confirmation (per spec §4.8). Прибрано дубль-кнопку Discard — resolves F2.4
  • Назва за замовчуванням Push A у demo. При Build from scratch — Workout · 2026-05-03 (per spec §4.1). При Repeat last — з джерела
  • Quick-add chips у кінці списку, поруч із + Add exercise — обидва способи додати в одній зоні, у thumb-reach коли список росте. Статичні в v1, без локального ranking-у (per spec §4.2)
  • Drag handle — 6 dots в 3 рядки по 2, як стандартна Material/iOS reorder affordance. Position зліва щоб thumb-reach, name-area по центру, ⋮ справа симетрично

Edge cases · related states

  • З суперсетом — builder-with-supersets.html
  • Empty list — header лишається, chips видимі (нижче empty-copy), замість карток — copy No exercises yet · Tap a chip below or "+ Add exercise" (per spec §4.7), Start workout disabled
  • Bodyweight вправа в Builder показує лише назву + м'язові групи; обробка ваги (isBodyweight → без kg-поля) — вже in-workout concern (per spec §5)
