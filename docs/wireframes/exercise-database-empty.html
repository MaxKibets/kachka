Opens from

  • Profile (profile.html) → DATA → Exercise database (push на Profile stack, Browse mode per spec §11.1)

Opens to

  • Tap + Create custom (sticky bottom) → exercise-create.html з порожнім name
  • Tap muscle filter chip → list filters by tag (per spec §11.4)
  • Tap search → field expands, system keyboard, case-insensitive substring match (per spec §11.3)
  • Tap system row → нічого не відбувається (per spec §11.1 — system exercises read-only, без detail у Browse mode)
  • Tap ← back → profile.html

Interactions

  • Search + filter sticky зверху під app-bar — лишаються видимі при скролі
  • Filter chips — single-select, sticky All першим (per spec §11.4)
  • System rows без ⋮ action button — read-only, ніщо не клікабельно крім візуального highlight
  • Bottom + Create custom sticky — завжди доступний (per spec §11.2)
