Opens from

  • In-workout · tap на kg / reps tile активного сета (in-workout.html)
  • In-workout з суперсетом — те саме (in-workout-with-supersets.html)

Opens to

  • Tap Save set → коміт значення в active set, sheet закривається. Якщо обидва поля заповнені і ✓ tapped — save + advance cursor + start rest. Label називає наслідок (set залогований), not generic "Done" (design-review F3.2, spec §7.2)
  • Swipe down (через grab handle) → закриває sheet без коміту
  • Tap on alternate field tab (REPS) → перемикає focus, quick-adjust ряд оновлюється на −5 −1 +1 +5 (per spec §7.2)

Interactions

  • Tap-to-edit per spec §7.3: значення select-all-нуте, перший tap на цифру замінює; backspace очищує
  • Quick-adjust ряд варіюється від field: kg = ±2.5 ±5, reps = ±1 ±5 (per spec §7.2)
  • Decimal separator (. чи ,) визначається locale, але внутрішньо завжди point (per spec §7.5)
  • Drag handle вгорі sheet-а — стандартний bottom-sheet swipe-to-dismiss

Edge cases · related states

  • Reps mode — quick-adjust перемикається. Можна показати в alt-варіанті (TBD)
  • Bodyweight вправа (per spec §7.4) — kg-поле сховане, тільки reps tab. Опц. +extra weight tab якщо юзер тицьнув
  • Failed reps — юзер набирає 0 і Save set. Сет вважається завершеним (per spec §5.7)
  • Locale-driven decimal separator — позиція . на key може бути ,. Поки показую . як default (en locale)

Notes

  • Sheet rendered як bottom-anchored (per visual §5.7 + spec §1: всі menus = bottom action sheets)
  • Backdrop затемнений ~60% (canvas з dim overlay) щоб in-workout context лишався читабельним але secondary
  • Field tabs — surface-2 default, surface-3 + 1.5px amber border коли active. Active value у amber, inactive у secondary text
  • Quick-adjust та digit keys — surface-2 з 0.5px subtle border, radius 10–12. Plex Mono Medium для цифр
  • Save set CTA — amber primary, full width, з check icon
  • Open question: чи показувати field tabs `kg` і `reps` як SEPARATE field readouts, чи об'єднувати в один великий read-out з перемикачем. Поки — два tab-fields. Перевага — обидва значення видно одразу, можна тапати між полями без закриття
  • Open question: ±0.5 кнопка для дробових інкрементів (gym plates 0.5kg) — поки не додаю, додамо в v1.x якщо буде запит
