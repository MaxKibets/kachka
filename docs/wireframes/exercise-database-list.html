Opens from

  • Profile (profile.html) → DATA → Exercise database (push на Profile stack, Browse mode per spec §11.1)

Opens to

  • Tap + Create custom (sticky bottom) → exercise-create.html з порожнім name
  • Tap custom row ⋮ → bottom action sheet: Edit / Delete (per spec §11.8) — sheet wireframe TBD
  • Tap custom row body → нічого (як system): окремого detail-екрана у v1 немає (per spec §11.8, detail → v2 §11.10)
  • Tap muscle filter chip → AND-фільтр з search (per spec §11.4)
  • Tap search → field expands, system keyboard, case-insensitive substring (per spec §11.3)
  • Tap system row → нічого (read-only, без detail у Browse mode per spec §11.1)
  • Tap ← back → profile.html

Interactions

  • Search + filter sticky зверху під app-bar
  • ⋮ button лише на custom rows. System rows без menu — read-only
  • Custom badge — маленький Mallard chip справа від назви для миттєвої idenfification
  • Sticky bottom + Create custom

Notes

  • Custom mark: inline badge Custom у Mallard tint справа від назви (identity) + ⋮ для дій (per spec §11.5). Chevron > свідомо НЕ використовуємо — у v1 немає detail-екрана для навігації, а badge explicit-ніший за тип і не плутається з navigation chevron-ом
  • System vs custom row affordance: custom має ⋮ справа (для edit/delete), system — нічого. Дві різні lights, але обидві read у Browse-режимі
  • Letter grouping (A, C, D, F, M, O, P, S) — алфавітний індекс. Custom і system mixed у тих самих секціях (на цьому фреймі не виділяємо custom-only список)
  • Showcase демонструє 3 custom + 9 system, mixed — Arnold press (A), Cable crossover (C), My weird squat (M, bodyweight)
  • Filter chips state — All активний (дефолт). Single-select per spec §11.4
  • Search input decorative; реальний impl — focus → keyboard → live filter
  • Soft-deleted custom не відображаються (per spec §11.9)

Edge cases · related states

  • Без жодного custom → exercise-database-empty.html
  • Search no-results → inline Create as 'X' affordance (per spec §11.6)
  • Filter Bodyweight-only — у v1 нема (per spec §11.10 — equipment filter відкладено в v2)
  • Custom ⋮ tap → bottom action sheet (Edit / Delete). Delete → confirmation per spec §1 — TBD як окремий мокап
