  • Swipe-down або backdrop tap → закриває sheet (поточний вибір уже застосований)

Interactions

  • Grab handle угорі — swipe down dismisses (spec §1 — bottom sheets)
  • Backdrop tap — закриває sheet (вибір уже застосовано instant)
  • Single-select — один активний row за раз. Маркер вибраного — тільки checkmark справа + mallard-тінт лейбла (design-review F5.2: прибрано дубль-маркер, була ще крапка зліва)

Notes

  • Один HTML файл показує два окремі sheets side-by-side для compact демонстрації. У реальному app це два незалежні bottom sheets, відкриваються по черзі з Profile root
  • Default selection — System у обох (per spec §12.2)
  • Language System auto-detect-ить локаль; якщо locale не EN і не UK — fallback English (per spec §12.2). Тут показано як "System (English)" — резолвлене значення показується в дужках для clarity
  • Чому bottom sheet а не push-screen для 3-row picker — узгоджено з §1 spec: усі action menus / pickers через bottom sheets. Економить навігаційну глибину для тривіальних виборів
  • Без footer (Cancel/Done прибрано, design-review F5.2): для 3-опційного picker instant-apply — нативний iOS-патерн; theme/language тривіально оборотні, commit-крок зайвий. Узгоджено зі spec §12.2
  • Висота sheet — content-driven (drag handle + title + 3 rows + actions). Якщо у v2 додасться більше опцій — sheet просто виростає

Edge cases · related states

  • OS тема змінилась поки app закритий + Theme=System → app перевідкривається в новій темі
  • System locale не EN/UK → Language sheet показує System (English) (fallback per spec §12.2)
  • Перемикання Language миттєве, без рестарту (per spec §12.2)
