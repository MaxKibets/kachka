Opens to

  • Tap system row → нічого (read-only, без detail screen per spec §11.8)
  • Tap custom row body → нічого (як system): окремого detail-екрана у v1 немає (per spec §11.8)
  • Tap custom ⋮ → bottom action sheet: Edit (→ custom creation sheet з prefill) + Delete (destructive, з confirmation). Єдина точка edit/delete у v1 (per spec §11.8)

Interactions

  • Sticky search + filter chips, скролиться список (як у Add mode)
  • Single-select muscle filter, sticky All першим і active by default (per spec §11.4)
  • Search reset на open
  • Свайп вправо від лівого краю — back до Profile (iOS gesture)

Notes

  • Header: ← back (push sub-screen, не modal). Title Exercise database — повний контекст де ми
  • Custom exercise differentiator: маленький Custom badge (mallard-dim background + mallard text, 10px uppercase) + ⋮ справа (per spec §11.5)
  • System exercises — без ⋮, без detail (read-only per spec §11)
  • Sample custom exercise: Pin squat, single-tag Legs (per spec §11 — custom має single-tag)
  • Soft-deleted custom exercises тут не відображаються (per spec §11.9)
  • Sort alphabetical (per spec §11.5) — Pin squat розташовується між Overhead press і Pull-up
