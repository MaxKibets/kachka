Opens from

  • Builder → row ⋮ на standalone exercise (builder.html, builder-with-supersets.html)

Opens to

  • Tap Add note → exercise note editor (exercise-note-sheet.html); note показується author hint-ом в Active workout (per spec §4.4)
  • Tap Add to superset → combined config sheet (multi-select партнерів + rounds/rest), створює нову групу (superset-config-sheet.html, per spec §6.2)
  • Tap Remove from workout → confirmation sheet (Remove this exercise?) → видаляє вправу з Builder
  • Свайп вниз або тап на scrim → close (per spec §1)

Interactions

  • Sheet відкривається після тапу ⋮ на row у Builder. Highlighted row у backdrop — амбер border на тій вправі, до якої прив'язаний sheet (visual cue для контексту)
  • Свайп вниз = close (per spec §1)
  • Тап поза sheet (на scrim) = close

Notes

  • Sheet header дублює контекст: name Bench press + subtitle Chest · Triceps · Shoulders — щоб юзер не плутався який саме exercise редагує. Sets у subtitle більше немає — у Builder сети не визначаються (per redesign)
  • Add to superset завжди створює нову групу (per spec §6.2 / §4.4 — combined sheet з multi-select партнерів). Ріст наявної групи — це Add exercise to group у меню самої групи (§6.7), не це standalone-row-меню. Тому group-letter shortcut (стара іконка A) прибрано: криптичний, дублював group-menu і не масштабувався на A/B/C
  • Reorder тут навмисно НЕ дублюється. Переміщення вправ — лише drag-handle зліва (per spec §4.6); меню composition-only. Move up/down прибрано: списки короткі, ручка зліва зроблена під one-hand thumb-drag, тож кнопки лише дублювали б жест і подовжували меню. A11y-reorder віддається на platform accessibility actions (невидимі custom actions на row), не на видимі рядки меню
  • Remove from workout — destructive (rust color --letter-e). Відділена від neutral actions тонким hairline-бордером (0.5px var(--border-divider)) + відступом, у стилі sheet-сімейства (per spec §1 — destructive внизу)
  • Row menu тепер composition-only per spec §4.4 (redesign, supersedes F2.1). Edit sets прибрано: редагування сетів переїхало в In-workout pending state (per spec §5.9), бо у Builder сети більше не визначаються. Add note лишається — author hint, що рендериться в Active workout; відкриває exercise note editor (exercise-note-sheet.html)
  • Assumption: Confirmation для Remove from workout per spec §1 — окремий confirmation sheet (TBD у Batch 3-4 — той самий компонент з Cancel вгорі і destructive внизу)

Edge cases · related states

  • Якщо вправа вже в групі (children A1/A2 у builder-with-supersets.html) — sheet адаптується: замість Add to superset з'являється Remove from group. Reorder в межах групи — теж drag (per spec §4.5 Reorder inside)
  • Group ⋮ menu — інший sheet (TBD): Edit rounds/rest, Add exercise to group, Reorder inside (drag), Ungroup (per spec §4.5)
  • Active workout row menu — спорідений компонент з додатковими actions (Skip, Insert after) per spec §5.5 — TBD у Batch 3
