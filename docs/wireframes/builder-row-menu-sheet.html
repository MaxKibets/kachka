Opens to

  • Tap Add to superset → партнер picker → superset-config-sheet.html
  • Якщо в workout-і вже є група A — letter chip A indicator. Tap → додає вправу до існуючої групи A (skip step partner picker, відкривається config sheet з prefilled candidates per spec §6.2)
  • Tap Move up / Move down → переміщує вправу в списку, sheet закривається
  • Tap Remove from workout → confirmation sheet (Remove this exercise?) → видаляє вправу з Builder
  • Свайп вниз або тап на scrim → close (per spec §1)

Interactions

  • Sheet відкривається після тапу ⋮ на row у Builder. Highlighted row у backdrop — амбер border на тій вправі, до якої прив'язаний sheet (visual cue для контексту)
  • Свайп вниз = close (per spec §1)
  • Тап поза sheet (на scrim) = close

Notes

  • Sheet header дублює контекст: name Bench press + subtitle 3 sets · Chest · Triceps · Shoulders — щоб юзер не плутався який саме exercise редагує
  • Letter chip A показано як indicator на Add to superset — означає що в workout-і вже є група A (per prompt). Якщо тап → додасть до неї. Якщо груп немає — chip відсутній, тап → партнер picker (нова група)
  • Move up / Move down — secondary affordance. Per spec §4.4 + §4.6 drag є primary. Корисно для accessibility (folks без drag), і фолбек коли drag важко (точність thumb)
  • Remove from workout — destructive (rust color --letter-e). Окрема divider-section з canvas-кольором візуально відмежовує destructive від neutral actions (per spec §1 — destructive внизу)
  • Per spec §4.4 повний menu також включає Edit sets, Add note — у v1 ставимо мінімум для Batch 2 wireframe. Можна розширити в наступних ітераціях
  • Assumption: Edit sets і Add note per spec §4.4 не included у цей wireframe (не зазначено в prompt-і). Якщо потрібно — додаємо у наступному раунді
  • Assumption: Confirmation для Remove from workout per spec §1 — окремий confirmation sheet (TBD у Batch 3-4 — той самий компонент з Cancel вгорі і destructive внизу)

Edge cases · related states

  • Якщо вправа вже в групі (children A1/A2 у builder-with-supersets.html) — sheet адаптується: замість Add to superset з'являється Remove from group; Move up/down в межах групи
  • Group ⋮ menu — інший sheet (TBD): Edit rounds/rest, Add exercise to group, Reorder inside, Move group up/down, Ungroup (per spec §4.5)
  • Active workout row menu — спорідений компонент з додатковими actions (Skip, Insert after) per spec §5.5 — TBD у Batch 3
  • Якщо в workout-і кілька груп (A, B) — chip indicator показує літеру першої групи, або show selector (TBD UX-варіант якщо понадобиться)
