Opens from

  • Tab bar → Profile з будь-якого таба
  • Today top-bar ⋯ → Settings (per spec §3.6 — short-cut на Profile root)

Opens to

  • Tap Theme або Language → settings-theme-language-sheet.html (bottom sheet picker)
  • Tap Show RPE / Rest haptic / Rest sound → toggle in place, без navigation
  • Tap Default rest → picker sheet з опціями (TBD — поки placeholder)
  • Tap Exercise database → exercise-database-list.html (Browse mode push на Profile stack)
  • Tap Backup & restore → backup-restore.html
  • Tap App version або Privacy → about.html
  • Tap GitHub → external link (open in browser)

Interactions

  • Скрол всього контенту, tab bar лишається фіксованим
  • Toggle-row тап анімує switch (mallard background, knob slide)
  • Picker-row чевроном вказує на відкриття bottom sheet

Notes

  • Per spec §12.1 — preferences inline на root, без nested settings menu. Грубовані секції замість окремих sub-screens
  • Section labels (PREFERENCES / WORKOUT / DATA / ABOUT) — uppercase 11px у text-tertiary, узгоджено з Builder section labels
  • Кожна row — 56px tall з icon prefix (Lucide-style line). Іконки 20px у text-secondary
  • Default rest у WORKOUT — поки placeholder. Spec згадує WORKOUT секцію але без конкретики default rest. Якщо не плануємо у v1 — прибрати в impl
  • App version показується як inline value (1.0.0 в Plex Mono), тап на ту ж row веде в about.html для повного списку
  • Privacy — окремий row, веде в About sub-screen з повним текстом privacy note
  • Без Delete all data у v1 (per spec §12.6)

Edge cases · related states

  • Show RPE OFF → бейджі @8 ховаються в History detail і set actions sheet (per spec §12.2)
  • Rest haptic + Rest sound обидва OFF → таймер просто візуально доходить до 0:00 без сигналів (per spec §12.3)
  • Theme System + OS у Light mode → app slips to Light theme (v1 — fallback, не пріоритет per visual §2)
