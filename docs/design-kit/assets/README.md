No logo or wordmark was provided in the source material. "Kachka" is
rendered in plain Manrope type wherever a mark would go (see the Brand
specimen card in the Design System tab). Do not invent, sketch, or
approximate a logo for this product.

## icons/

11 SVGs, extracted verbatim (same path data, unmodified) from
`WorkoutScreen.dc.html` and `RestBar.dc.html` — this is the product's
**complete** icon vocabulary as evidenced in the source; nothing here was
invented. All are 24×24, `currentColor`, stroke-width 2.4 with round
caps/joins — except `more-horizontal.svg`, which is filled dots, not a
stroke icon.
